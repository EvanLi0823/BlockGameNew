## 使用建议

### 修改 Prefab 前的工作流

1. **查看索引** - 确认 Prefab 名称和复杂度
2. **阅读文档** - 了解完整的 GameObject 层级结构
3. **查找UI元素** - 使用 grep 定位特定元素
4. **修改 Prefab** - 在 Unity 中进行修改
5. **重新生成文档** - 保持文档同步

### 批量操作

**批量搜索特定元素**：
```bash
# 查找所有包含 "Button" 组件的 GameObject
grep -h "MonoBehaviour, MonoBehaviour" references/Prefab_*.md | sort | uniq
```

**统计所有 Prefab 的总 GameObject 数**：
```bash
grep -h "GameObject总数:" references/Prefab_*.md | awk -F': ' '{sum+=$2} END {print sum}'
```
